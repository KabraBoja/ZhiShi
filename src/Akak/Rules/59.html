Приставка + наречие: запросто навсегда послезавтра извне. Приставки на-, в- + собирательное числительное: надвое вдвое втроем НО: по двое, по трое. Приставка + местоимение, полн. прилагат. (не начинающ. с гласной) вничью впрочем зачастую. Приставка + краткое прилагательное: досуха набело. Приставка + существительное, где между приставкой и существительным нельзя вставить определение: вброд исподтишка
